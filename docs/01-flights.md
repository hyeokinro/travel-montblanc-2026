## CPH 환승 정보

### 가는편 환승: 2시간 15분 (06:00 착 → 08:15 발)
- 쉥겐 입국심사가 CPH에서 진행됨 (비EU → 쉥겐 첫 진입점)
- CPH → GVA는 쉥겐 내부 이동이라 추가 심사 없음
- WCHR 신청 시 직원이 입국심사 + 게이트까지 에스코트

### 귀국편 환승: 1시간 50분 (22:05 착 → 23:55 발)
- 쉥겐 출국심사가 CPH에서 진행
- **가장 긴장되는 구간** — WCHR + 아기 + 카시트(기내반입) 동시 이동
- SAS 동일 예약이라 미스커넥션 시 항공사 책임

### 장모님 단독 귀국 (6/3)
- GVA → CPH → ICN, WCHR 인계
- 영어 불가하지만 WCHR 직원이 게이트까지 안내

### 어머니/이모 도착 (6/6 GVA 10:20)
- CPH에서 쉥겐 입국심사 (WCHR 직원 안내)
- 영어 불가 — WCHR이 사실상 필수

## SAS 기내 규정

### 카시트
- 좌석 팔걸이 간 41~43cm (16~17in), 앞뒤 38cm (15in) 이내 기내반입 가능
- 서안이 별도 좌석 구매 → 카시트 기내 사용 (앞보기, 항공벨트 고정)
- CPH에서도 카시트 수령 가능 (환승 시 직접 들고 이동)

### BedBox
- SAS에서 사용 가능 확인됨

### 유모차
- 위탁수하물 처리, 크기 제한 없음
- 보호용 비닐백 필요 (공항에서 구매 가능)
- 게이트 체크도 가능 (접이식 유모차)

### WCHR (Wheelchair)
- 게이트까지 휠체어 이동, 환승 공항 및 도착 수하물까지 커버
- 기내용 가방 도움도 제공
- 장모님(ZBTOB2) + 어머니/이모(YTUQGK) 모두 신청 완료
- 걷는 데 지장 없어도 장거리 보행 + 비영어권 + 대형 허브 환승이라 적절한 사용

### 좌석
- SAS는 아이와 부모 인접좌석 보장

## 공항별 우대레인

### 인천공항 (ICN)
- **7세 미만 동반 → Priority Lane** (보안검색 + 출입국심사)
- 동반 가족 최대 3명까지 함께 입장
- **체크인 카운터에서 Priority Lane Pass 요청** 필요

### 코펜하겐공항 (CPH)
- **6세 미만 동반 → Family Lane** (보안검색)
- 2세 미만 아동/유모차: 보딩패스 스캔 전 **패밀리 아이콘 찾아서 수동 입구**로 진입
- WCHR 이용 시 직원이 Family Lane까지 안내

### 제네바공항 (GVA)
- 공식 가족 전용 우대레인 없음
- 소규모 공항이라 보안 대기 보통 짧음
- 유료 Priority Lane은 있으나 불필요할 가능성 높음

## 아기 기내 관련 참고

- 유아식: SAS는 퓨레 미제공. 직접 준비 필요
- 액체류: 영아용 우유/이유식은 100ml 제한 면제 (합리적인 양)
- 기내 기저귀 교환: 모든 화장실에 교환대 있음
- 바시넷: 장거리편에서 9개월 미만/11kg 이하 사용 가능하나, 서안이는 해당 안 됨 (14개월)
