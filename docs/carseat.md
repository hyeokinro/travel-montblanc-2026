# 🚼 카시트 설치 가이드

[← 메인으로 돌아가기](../README.md)

---

**카시트**: Evenflo Tribute™ Convertible (FMVSS 213 + 항공기 인증)
**아기**: S (13개월, ~11kg)
**핵심 원칙**: 차량에서는 반드시 **뒤보기**, 비행기에서는 **앞보기**

---

## 용어 정리

| 쉬운 말 | 영어 (매뉴얼) | 설명 |
|---------|-------------|------|
| 발판 | Recline Stand | 카시트 밑에 달린 접이식 발. 앞/뒤로 접어 각도 조절 |
| 벨트 구멍 | Belt path | 카시트 옆면의 벨트 통로. 앞면(뒤보기용) / 뒷면(앞보기용) |
| 어깨끈 슬롯 | Harness slot | 어깨끈이 나오는 구멍 4줄. 아래3개=뒤보기, 위3개=앞보기 |
| 가슴 클립 | Chest clip | 양쪽 어깨끈 연결 클립. 겨드랑이 높이에 맞추기 |
| 버클 | Buckle | 가랑이 사이 빨간 버튼 결합부 |
| ISOFIX / LATCH | Lower Anchor Connectors | 카시트 뒤의 줄+훅 → 차량 좌석 철봉에 걸어 고정 |
| 수평 화살표 | Level arrow | 카시트 옆면 화살표. 뒤보기 시 바닥과 수평이어야 정상 |

## 1단계: 테슬라 → 인천공항 (뒤보기 + ISOFIX)

1. **발판을 앞쪽으로 접기** — 카시트를 들어서 밑의 발판을 앞으로 접는다
   *(Flip recline stand toward front to recline position — p.18)*

2. **어깨끈을 아래쪽 구멍에 끼우기** — 4줄 중 아래 3개에서 어깨 높이에 가장 가까운 것
   *(Harness straps in one of 3 lower slots, at or below shoulders — p.20)*

3. **앞면 천 벗기고, 줄+훅을 앞면 벨트 구멍으로 통과** — 시트패드 벗기면 양쪽에 벨트 통로. 카시트 뒤의 줄+훅을 이 통로로 통과
   *(Route LATCH strap through rear-facing belt path — p.21)*

4. **차량 좌석 철봉에 훅 걸기** — 뒷좌석 쿠션 사이 철봉(ISOFIX bar)에 좌우 훅 각각 걸고 세게 당겨보기
   *(Attach hooks to lower anchor bars, pull hard to confirm — p.21)*

5. **몸으로 눌러서 줄 조이기** — 체중 실어 누르면서 LATCH 줄 당겨 조임. **좌우/전후 2.5cm 이상 움직이면 안 됨**
   *(Push down with full weight, pull strap tight. Must not move >25mm — p.22)*

6. **수평 화살표 확인** — 바닥과 수평이어야 정상. 기울어져 있으면 **앞쪽 아래에 말은 수건** 끼우기
   *(Level arrow must be parallel to ground. Use rolled towel if needed — p.18)*

> ⚠️ 위쪽 슬롯(맨 위)은 뒤보기에 금지 *(Do NOT use upper harness slots rear-facing)*
> ⚠️ ISOFIX와 시트벨트 동시 사용 금지 *(Never use LATCH and seat belt together)*

### 수건 받치기
카시트를 뒤보기로 설치 후 옆면의 **수평 화살표(Level Arrow)**가 바닥과 수평이어야 해요.

차량 좌석마다 기울기가 달라서 발판만으로 수평이 안 될 수 있어요. **카시트가 너무 세워지면 아기 머리가 앞으로 꺾여 기도가 좁아질 수 있어요.**

→ **카시트 앞쪽(발 쪽) 아래에 단단히 말은 수건을 받쳐** 앞을 살짝 들어올리면 뒤가 눕혀져서 수평에 가까워짐.

- 테슬라 모델3과 푸조 5008 좌석 각도가 다르니 **둘 다 설치 후 확인**
- 수건 하나 차에 항상 준비

## 2단계: SAS 비행기 (앞보기 + 항공벨트)

1. **발판을 뒤쪽으로 접기** — 카시트가 꼿꼿하게 서도록
   *(Flip recline stand toward rear to upright position — p.30)*

2. **어깨끈을 위쪽 구멍으로 옮기기** — 위 3개 중 어깨에 가장 가까운 것
   *(Harness straps in one of 3 upper slots, at or above shoulders — p.31, 35)*

3. **항공기 좌석 완전히 세우고, 카시트를 앞보기로 배치**
   *(Aircraft seat fully upright, place car seat forward-facing — p.49)*

4. **벨트를 뒷면 구멍으로 통과** — 뒷면 시트패드 들추면 벨트 통로. 항공 벨트를 한쪽→반대쪽
   *(Thread aircraft belt through forward-facing belt path — p.49)*

5. **버클 채우고 벨트 최대한 조이기**
   *(Fasten belt buckle and pull tight)*

6. **여분 벨트를 다시 구멍으로 통과시켜 추가 조임** — 몸 실어 누르면서 당김. **2.5cm 이상 움직이면 재조임**
   *(Route excess belt back through belt path, push down and tighten — p.50)*

> ⚠️ 항공기에서 ISOFIX/LATCH 절대 금지 *(Never use LATCH on aircraft)*
> ⚠️ 아래쪽 슬롯(맨 아래)은 앞보기에 금지 *(Do NOT use lowest slot forward-facing)*

### 참고
- 버클이 등 뒤에 오면 작은 베개/담요로 쿠션 (p.50)
- **CPH 환승**: 카시트 분리 → 들고 이동 → 재설치 (1시간 50분)
- 승무원에게 매뉴얼 보여주면 도움 가능 *Show this manual to cabin crew*

## 3단계: 푸조 5008 (뒤보기 + ISOFIX)

**1단계와 완전히 동일한 방법.**

### 푸조 참고
- ISOFIX는 **2열 양쪽 외측에만** 있음 (가운데 없음)
- 유럽식 ISOFIX와 LATCH 훅 호환됨
- 수건 받치기: 설치 후 수평 화살표 확인 (테슬라와 각도 다를 수 있음)

### 13일간 팁
- 카시트 고정해두고 S만 태우고 내리기
- 매회 어깨끈 조임 확인 (어깨에서 손가락으로 잡혀보면 안 됨)
- **여름 주차 시** 금속 부분 화상 주의 → 밝은색 담요로 덮기

---

## 4단계: 귀국 (6/11)

1. 푸조에서 카시트 분리
2. GVA에서 가족 하차 → H 반납 → 셔틀 복귀
3. 카시트 들고 탑승 (기내 반입)
4. 기내: **2단계와 동일**
5. CPH 환승: 분리 → 들고 이동 → 재설치 (1시간 50분) ⚠️
6. ICN → 테슬라: **1단계와 동일**

## 매회 S 태울 때 체크

1. ✅ 등+엉덩이가 바닥/등받이에 평평하게 밀착 *(p.41)*
2. ✅ 어깨끈 → 버클 딩 소리까지 끼우기 *(p.41)*
3. ✅ 앞면 아래 줄 당겨 어깨끈 조임. 어깨에서 **엄지로 잡혀보면 안 됨** *(p.43)*
4. ✅ 가슴 클립 → 겨드랑이 높이에 딸칵 *(p.42)*
5. ✅ 가랑이 벨트 밀착. 엉덩이 아래로 가면 절대 안 됨 *(p.42)*
