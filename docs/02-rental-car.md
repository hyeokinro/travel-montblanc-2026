## 차량 정보 (푸조 5008 G26)

### 좌석 배치
```
[1열] 운전석 — 조수석
[2열] 외측(ISOFIX) — 가운데(ISOFIX없음) — 외측(ISOFIX)
[3열] 접이식 2석 (평상시 접어서 트렁크 사용)
```

### 적재 공간
- 3열 접었을 때: **916L** → 캐리어(28"×1 + 25"×2~3) + 유모차 충분
- 전석 접었을 때: 2,232L

### 카시트 설치
- 2열 ISOFIX는 **양쪽 외측 좌석에만** 있음 (가운데 없음)
- 이븐플로 트리뷰트 (폭 43cm): 컴팩트하여 외측 ISOFIX 또는 시트벨트 설치 모두 가능
- **뒤보기(rear-facing)로 설치** — 만 2세 미만이므로 필수
- 상세: [카시트 설치 가이드](07-carseat.md)

### 어른 합류 기간 (6/6~11)
- 최대 성인 4명 + 아기
- 1열: 2명 / 2열: 카시트 + 성인 2명 / 3열: 미사용
