### CPH 환승

| 항목 | 가는편 | 귀국편 |
|------|--------|--------|
| 환승시간 | 2시간 15분 | **1시간 50분** ⚠️ |
| 입국/출국심사 | CPH에서 쉥겐 입국심사 | CPH에서 쉥겐 출국심사 |
| CPH→GVA | 쉥겐 내부 (추가심사 없음) | — |
| WCHR | 직원이 심사+게이트 에스코트 | 동일 |

> **CPH는 쉥겐 입국/출국 포인트**. GVA(스위스)도 쉥겐이라 CPH↔GVA는 국내선 취급.
> GVA→프랑스 국경도 자유통과 (검문 없음).

### 공항 우대레인

| 공항 | 우대 | 방법 |
|------|------|------|
| **ICN** | 7세 미만 동반 Priority Lane | 체크인 카운터에서 패스 요청 (동반 3인까지) |
| **CPH** | 6세 미만 동반 Family Lane | 바닥 패밀리 아이콘 → 수동 입구 진입 |
| **GVA** | 별도 무료 우대 없음 | 소규모 공항, 대기 짧음 |

> S(14개월) 동반이므로 어떤 공항이든 가족 우선레인 자격.
> 단독 환승 시에는 WCHR로 게이트 투 게이트.

### SAS 기내 규정

| 항목 | 규정 |
|------|------|
| 카시트 | 팔걸이 간 41~43cm 이내 → 기내반입 OK. CPH 환승 시 직접 들고 이동 |
| BedBox | 사용 가능 |
| 유모차 | 위탁 (크기 무제한). 비닐백 필요 (공항 구매 가능). 게이트체크도 가능 |
| WCHR | 게이트+환승+도착 수하물까지 커버 |
| 좌석 | 아이+부모 인접 보장 |
| 유아식 | SAS 퓨레 미제공 → 직접 준비. 영아용 액체 100ml 면제 |
| 바시넷 | 9개월 미만/11kg 이하 → S는 해당 안 됨 |

### 픽업 절차 (5/29)

| 순서 | 할 일 |
|------|------|
| 1 | GVA 도착 → 수하물 찾기 |
| 2 | WhatsApp으로 MDS에 연락 → 프랑스 섹터에서 대기 |

**필수 지참**: 여권, 국제운전면허증, 신용카드

**대기 시간**: 비행편 기반 → 착륙 후 최대 1시간 30분 / 예약 기반 → 약속 후 30분

### 픽업 시 수령 (반납 시 모두 반납!)

| 수령물 | 비고 |
|--------|------|
| 차량 열쇠 **2개** | ⚠️ **분산 보관** (1개는 H, 1개는 J) |
| 자동차 등록증 (Grey Card) | 항상 차량에 비치. 국경 통과 시 필요 |
| 보험증서 | 차량에 비치 |

> ⚠️ 열쇠/등록증 미반납 시 카드로 청구됨. 분실 시 국경 통과 불가한 나라도 있음.

### 연료

- 픽업 시 **약 10L만** 들어있음 (약 100km 주행 가능) → **첫 주유소에서 바로 주유**
- 반납 시에는 **채울 필요 없음** — 최대한 사용 후 그냥 반납
- 연료: **Sans Plomb 95 (초록색 노즐)** — 디젤(Gazole) 절대 금지

**반납센터 주소**: 차량 GPS에 사전 입력되어 있음

**반납 시 제출물**: 차량 열쇠 2개 + 자동차 등록증 + 보험증서 → **모두 반납**

### 프랑스 섹터 통과 규칙
- 렌터카 계약서 + 탑승권 **모두 소지** 필요 (국제→프랑스 섹터 이동 시)
- GVA 프랑스 측 출입구 **22시 폐쇄** — 6/11 귀국편 20:05이므로 여유 있음

### 스위스 비네트
- **불필요** ❌ — 전 구간 프랑스 A40 고속도로만 이용

### 환경스티커 (Crit'air)
- 리스차량은 **부착 불필요** ✅ (푸조리스 공식 안내)
- 제네바 시내 관광도 일반도로로 진입 가능
- 프랑스 A40 톨비만 해당 (편도 ~€6~8)

### 차량 좌석/적재

```
[1열] 운전석 — 조수석
[2열] 외측(ISOFIX) — 가운데(없음) — 외측(ISOFIX)
[3열] 접이식 2석
```

- 3열 접으면 916L → 캐리어+유모차 충분
- 어른 합류(6/6~11): 성인4+S → 1열2 + 2열(카시트+2) + 3열 미사용
- 카시트: 2열 외측 ISOFIX에 **뒤보기** 설치

### 주유
- **무연 휘발유**: Sans Plomb / SP95 또는 SP98
- 디젤(Gazole) 넣지 않도록 주의
- 셀프, 카드 결제
