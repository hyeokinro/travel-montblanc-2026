# 🥩 식료품 구매 가이드

[← 메인으로 돌아가기](../)

> 프랑스 까르푸에서 장볼 때 참고. 한국과 부위명/명칭이 달라서 정리해둡니다.
> 까르푸 정육 코너에서 패키지에 적힌 프랑스어를 보고 찾으세요.

## 🐂 소고기 (Bœuf / 뵈프)

### 한국 요리용 추천

| 한국 부위 | 프랑스어 | 발음 | 용도 | 비고 |
|-----------|---------|------|------|------|
| **등심** | Faux-filet | 포필레 | 스테이크, 구이 | 한국 등심과 가장 비슷. 마블링 있음 |
| **안심** | Filet | 필레 | 스테이크 | 가장 부드러운 부위. 비쌈 |
| **채끝** | Rumsteck | 럼스텍 | 스테이크, 로스트 | 담백하고 lean |
| **꽃등심/립아이** | Entrecôte | 앙트르꼬뜨 | 스테이크, 구이 | 지방 풍부, 풍미 좋음 |
| **국거리/불고기용** | Bavette | 바베뜨 | 볶음, 불고기 | 얇게 썰어달라고 하면 불고기용 OK |
| **찌개/국용** | Bourguignon (덩어리) | 부르기뇽 | 찌개, 조림 | 미리 깍둑 썰어진 것. 저렴 |
| **사골/곰탕** | Os à moelle | 오 아 무알 | 국물 | 사골 대용. 정육 코너에서 요청 |
| **갈비** | Plat de côtes | 플라 드 꼬뜨 | 찜, 탕 | 한국 갈비와 다르지만 갈비찜 가능 |
| **차돌박이** | Poitrine (de bœuf) | 뿌아트린 | 국, 구이 | 지방층. 얇게 썰린 건 드묾 |

> 💡 **불고기/잡채용 팁**: "Bavette"를 사서 **"tranché fin"(얇게 썰어주세요)**이라고 요청하거나, 냉동 후 반냉동 상태에서 직접 얇게 썰기.

> 💡 까르푸 패키지에 **"à griller"** = 구이용, **"à braiser"** = 조림/찜용, **"à mijoter"** = 끓이기용, **"haché"** = 다진 고기

## 🐷 돼지고기 (Porc / 뽀르)

| 한국 부위 | 프랑스어 | 발음 | 용도 | 비고 |
|-----------|---------|------|------|------|
| **삼겹살** | Poitrine de porc | 뿌아트린 드 뽀르 | 구이, 김치찜 | 통째 또는 슬라이스. 한국 삼겹처럼 얇게 안 썰려있음 |
| **목살** | Échine | 에신 | 구이, 불고기 | 어깨~목 사이. 지방 적당 |
| **안심** | Filet mignon de porc | 필레 미뇽 드 뽀르 | 구이, 돈까스 | 부드럽고 lean |
| **등심** | Côte de porc / Longe | 꼬뜨 드 뽀르 | 돈까스, 구이 | 뼈붙이(côte) / 뼈없는(longe) |
| **앞다리** | Épaule | 에뽈 | 조림, 찌개 | 살이 많고 저렴 |
| **뒷다리(후지)** | Jambon (cru/cuit) | 장봉 | 로스트, 햄 | cru=생, cuit=익힌 것(ham) |
| **다진돼지** | Haché de porc | 아셰 드 뽀르 | 만두, 완자 | |
| **베이컨** | Lardons | 라르동 | 볶음, 찌개 | 한국 베이컨과 다름. 두껍고 짠 편. nature=무염, fumé=훈제 |

> 💡 **삼겹살 팁**: 프랑스 삼겹은 한국처럼 얇게 안 썰려있어요. 통으로 사서 냉동 후 반냉동에서 직접 썰거나, 정육코너에서 **"tranché fin"** 요청.

> 💡 **제육볶음/김치찌개용**: Échine(목살) 또는 Poitrine(삼겹) 추천

## 🐔 닭고기 (Poulet / 뿔레)

| 한국 부위 | 프랑스어 | 발음 | 용도 | 비고 |
|-----------|---------|------|------|------|
| **통닭** | Poulet entier | 뿔레 앙띠에 | 삼계탕, 백숙 | 한국보다 큼 (보통 1.5~2kg) |
| **닭가슴살** | Blanc de poulet / Filet | 블랑 드 뿔레 | 볶음, 샐러드 | |
| **닭다리** | Cuisse de poulet | 뀌스 드 뿔레 | 찜, 구이, 조림 | 윗다리+아랫다리 통째 |
| **닭날개** | Ailes de poulet | 엘 드 뿔레 | 튀김, 구이 | |
| **닭볶음탕용** | Poulet découpé | 뿔레 데꾸뻬 | 닭볶음탕 | 미리 잘라진 것 |
| **뼈/육수용** | Carcasse de poulet | 카르카스 | 국물 | 정육코너에서 요청. 매우 저렴 |

> 💡 **삼계탕**: 통닭(poulet entier)을 사서 직접 요리. 인삼/대추/찹쌀은 한국에서 가져가기.

## 🛒 까르푸 정육코너 실전 팁

### 패키지 라벨 읽기

| 표기 | 의미 |
|------|------|
| **VBF** | Viande Bovine Française = 프랑스산 소고기 |
| **VPF** | Viande Porcine Française = 프랑스산 돼지고기 |
| **Bio** / **AB** | 유기농 |
| **Élevé en plein air** | 방목 |
| **Haché** | 다진 고기 |
| **Tranché** | 슬라이스 |
| **À griller** | 구이용 |
| **À braiser** | 조림/찜용 |
| **À mijoter** | 끓이기/국용 |
| **À rôtir** | 오븐 로스트용 |
| **Sous vide** | 진공포장 |

### 정육코너에서 유용한 프랑스어

| 한국어 | 프랑스어 | 발음 |
|--------|---------|------|
| 얇게 썰어주세요 | Tranché fin, s'il vous plaît | 트랑셰 팽 실부플레 |
| 두껍게 썰어주세요 | Tranché épais, s'il vous plaît | 트랑셰 에페 실부플레 |
| 500g 주세요 | Cinq cents grammes, s'il vous plaît | 생 쏭 그람 실부플레 |
| 뼈 빼주세요 | Désossé, s'il vous plaît | 데조세 실부플레 |
| 이거 주세요 | Celui-ci, s'il vous plaît | 슬뤼시 실부플레 |

> 💡 까르푸 정육코너 운영시간: 보통 오전에 신선. 포장육은 종일 구매 가능.
> 포장육 라벨의 부위명을 이 페이지에서 찾아보면 됩니다.

## 🧀 기타 유용한 식재료

| 한국어 | 프랑스어 | 비고 |
|--------|---------|------|
| 계란 | Œufs | 까르푸에 다양 (bio=유기농, plein air=방목) |
| 우유 | Lait | entier=전지, demi-écrémé=저지방, écrémé=무지방 |
| 버터 | Beurre | doux=무염, demi-sel=가염 |
| 식용유 | Huile | tournesol=해바라기(가장 흔한), olive=올리브 |
| 밀가루 | Farine | T45=박력분, T55=중력분, T65=강력분 |
| 간장 | Sauce soja | 아시안 코너 |
| 식초 | Vinaigre | de vin=와인식초, de cidre=사과식초 |
| 소금 | Sel | fin=고운 소금, gros=굵은 소금 |
| 후추 | Poivre | |
| 쌀 | Riz | 아시안 코너에 Riz rond(둥근쌀) → 한국 쌀에 가장 비슷 |

> 💡 **쌀 팁**: 까르푸 아시안 코너에서 **Riz rond** 또는 **Riz à sushi**를 사면 한국 쌀과 가장 비슷해요. 일반 Riz long(장립종)은 밥 짓기에 부적합.
