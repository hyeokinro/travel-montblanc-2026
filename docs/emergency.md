## 날씨 (5월 말~6월 초)

| 항목 | 예상 |
|------|------|
| 낮 | 15~22°C |
| 밤 | 5~10°C |
| 고산(에귀뒤미디 3,842m) | -5~5°C |
| 비 | 간헐적 소나기 |
| 일몰 | ~21:00 |

### 복장
- 레이어링 필수 (아침 시원 → 낮 따뜻 → 저녁 선선)
- 에귀 뒤 미디: 패딩/플리스 + 선글라스 + 선크림
- 호수/마을: 가벼운 긴팔 + 바람막이
- S: 겹겹이, 모자, 자외선 차단
