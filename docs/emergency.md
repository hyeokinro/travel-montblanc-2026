## 🚗 프랑스/스위스 운전 가이드

### 한국과 다른 점 (꼭 숙지)

| 항목 | 내용 |
|------|------|
| **우선순위** | 로터리(rond-point)에서는 **이미 진입한 차가 우선**. 한국과 같음. 단, 일부 소도시에서 **우측 우선(priorité à droite)** 표지판 있으면 오른쪽에서 오는 차에 양보 |
| **라운드어바웃** | 몽블랑 밸리에 로터리가 매우 많음. 진입 전 좌측 확인 → 진입 → 나갈 출구 전 우측 깜빡이 |
| **제한속도 단속** | 프랑스는 과속 카메라(radar) 매우 많음. **렌터카로 딱지 맞으면 카드에서 자동 차감**됨 |
| **음주운전** | 혈중 알코올 0.05% (한국 0.03%보다 높지만, 와인 한 잔도 위험). **렌터카라 절대 금주 운전** |
| **전조등** | 프랑스는 낮에도 주간주행등 필수 (신형 차라 자동으로 켜짐) |
| **휴대폰** | 운전 중 휴대폰 손에 들면 €135 벌금. 블루투스/거치대 사용 |
| **어린이 좌석** | 프랑스는 만 10세 미만 뒷좌석 의무. 카시트 필수 (만 4세까지). 우리는 해당 ✅ |
| **경적** | 시내에서 불필요한 경적 사용 금지 (€35 벌금) |
| **비상조끼** | 차량 내 형광 조끼(gilet jaune) 비치 의무. 렌터카에 보통 있으나 확인 필요 |
| **삼각대** | 고장/사고 시 차량 뒤 30m에 삼각대 설치 의무. 렌터카에 보통 포함 |

### 속도 제한

| 도로 | 맑은 날 | 비 올 때 |
|------|---------|---------|
| 고속도로 (Autoroute) | 130 km/h | 110 km/h |
| 일반도로 | 80 km/h | 80 km/h |
| 시내 | 50 km/h | 50 km/h |
| 주거지역 (Zone 30) | 30 km/h | 30 km/h |

> ⚠️ 프랑스 과속 카메라는 **1km/h 초과부터 단속**. 속도계보다 5~10 낮게 달리는 습관 권장.

### 고속도로 톨비

- A40 (White Motorway): 편도 ~€6~8
- 톨부스에서 **티켓 뽑고 → 출구에서 결제** (카드/현금)
- "t" 표시가 있는 레인 = 카드 가능
- 무인 레인에서 카드 안 먹히면 인터콤 버튼 누르기

### 몽블랑 터널

- 편도 ~€50
- 숙소에서 35분
- **일요일 오후 이탈리아→프랑스 방향 정체** 가능 → 오전 출발 권장

- 아기가 차멀미할 수 있으니 구불길 전 환기

## 🚨 차량 사고/고장 시

### 푸조 어시스턴트 (24시간)

| 구분 | 번호 |
|------|------|
| 프랑스에서 | 01 47 89 24 24 |
| 프랑스 외 | +33 1 47 89 24 24 |
| 한국 로밍폰 | 001 33 1 47 89 24 24 |
| 이메일 | das@ima.eu |

> 전화 연결 후 **1번 = 영어**, 2번 = 불어

### 전화 시 필요한 정보
- 차량번호판
- 계약자 성함 + 연락 가능한 전화번호
- 현재 위치 주소
- 픽업일/반납일/반납장소

### 사고 유형별 대응

| 상황 | 조치 |
|------|------|
| **혼자 사고** | 자필 사고 경위서 작성 (6하원칙) → 어시스턴트 전화 |
| **상대방 있는 사고** | 사고합의서(Accident Report) 작성 후 한 장씩 보관 → 어시스턴트 전화 |
| **차량 이동 불가** | 어시스턴트 전화 → 견인(1~3시간) → 수리 1일 이상 시 렌터카 지원 |
| **유리/타이어 파손** | 어시스턴트 전화 → **선승인** 받고 → 수리 → 영수증+경찰진술서 원본 보관 |
| **도난** | 경찰 신고(필수!) → 어시스턴트 전화 |

> ⚠️ 어시스턴트 **사전 동의 없이** 수리하면 환급 불가!
> ⚠️ 영수증 원본 + 경찰진술서 원본 없으면 보상 불가!

### 보험 적용 안 되는 것
- 차량 내 개인물건 분실/도난
- 열쇠/등록증 분실
- 연료 고갈
- 과속/음주 관련 사고
- 범칙금, 식사비, 전화비, 톨비

### 범칙금 참고
- 과속/주차위반 범칙금은 **한국 집 주소로 직접 발송** (15일~3개월 후)
- 리스차량은 계약자 명의 등록이라 회피 불가
- 장기 여행 시 집에 우편물 확인 요청해두기

## ⚠️ 유럽 처음 방문 시 주의사항

### 하지 말아야 할 것들

| 상황 | 주의 |
|------|------|
| **식당에서 큰 소리** | 유럽 식당은 조용한 편. 한국식 큰 목소리 주의. 특히 저녁식사 |
| **웨이터 부르기** | 손 들거나 "Excusez-moi" — 한국식으로 "여기요!" 크게 부르지 않기 |
| **팁** | 프랑스는 서비스료 포함(service compris). 팁 의무 아님. 만족하면 잔돈 남기는 정도 |
| **계산** | 테이블에서 계산. 한국처럼 카운터에 가지 않기. "L'addition, s'il vous plaît" |
| **사진 찍기** | 낯선 사람(특히 아이) 얼굴 촬영 주의. 프랑스는 초상권 엄격 |
| **매장 입장** | 작은 가게 들어갈 때 **"Bonjour"** 인사 필수. 안 하면 무례하게 느껴짐 |
| **가게 퇴장** | 나올 때 **"Merci, au revoir"** — 안 사도 인사 |
| **줄 서기** | 새치기 절대 금지. 프랑스인들이 매우 민감해하는 부분 |
| **물 주문** | 식당에서 수돗물은 무료 "Une carafe d'eau". 생수 시키면 유료 (€3~5) |
| **일요일** | 대부분 상점 닫힘. 레스토랑/카페/관광지는 영업. 마트는 오전만(까르푸 일 09~12:30) |
| **낮잠 시간** | 점심~14시경 소도시는 가게가 닫기도 함 |
| **현금** | 소액도 카드 OK (프랑스는 카드 사회). 단 작은 마을 노점/마르셰는 현금 필요할 수 있음 |
| **화장실** | 무료 공중화장실 적음. 카페/레스토랑 이용하면서 사용. 고속도로 휴게소는 무료 |

### 알아두면 좋은 프랑스어 (부모님용)

| 한국어 | 프랑스어 | 발음 |
|--------|---------|------|
| 안녕하세요 | Bonjour | 봉주르 |
| 감사합니다 | Merci | 메르시 |
| 안녕히 계세요 | Au revoir | 오 르부아르 |
| 실례합니다 | Excusez-moi | 엑스퀴제 무아 |
| 네 / 아니오 | Oui / Non | 위 / 농 |
| 얼마예요? | C'est combien ? | 세 콩비엥? |
| 계산서 주세요 | L'addition, s'il vous plaît | 라디시옹 실부플레 |
| 물 주세요 | De l'eau, s'il vous plaît | 들로 실부플레 |
| 화장실 어디예요? | Où sont les toilettes ? | 우 송 레 뚜알렛? |
| 도와주세요 | Aidez-moi | 에데 무아 |

## 날씨 (5월 말~6월 초)

| 항목 | 예상 |
|------|------|
| 낮 | 15~22°C |
| 밤 | 5~10°C |

### 복장
- 레이어링 필수 (아침 시원 → 낮 따뜻 → 저녁 선선)
- 에귀 뒤 미디: 패딩/플리스 + 선글라스 + 선크림
- 호수/마을: 가벼운 긴팔 + 바람막이
- S: 겹겹이, 모자, 자외선 차단
