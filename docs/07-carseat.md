# 🚼 카시트 설치 가이드

[← 메인으로 돌아가기](../README.md)

---

**카시트**: Evenflo Tribute™ Convertible (FMVSS 213 + 항공기 인증)  
**아기**: 서안 (13개월, ~11kg)  
**핵심 원칙**: 차량에서는 반드시 **뒤보기**, 비행기에서는 **앞보기**

---

## 용어 정리

| 쉬운 말 | 영어 (매뉴얼) | 설명 |
|---------|-------------|------|
| 발판 | Recline Stand | 카시트 밑에 달린 접이식 발. 앞/뒤로 접어 각도 조절 |
| 벨트 구멍 | Belt path | 카시트 옆면의 벨트 통로. 앞면(뒤보기용) / 뒷면(앞보기용) |
| 어깨끈 슬롯 | Harness slot | 어깨끈이 나오는 구멍 4줄. 아래3개=뒤보기, 위3개=앞보기 |
| 가슴 클립 | Chest clip | 양쪽 어깨끈 연결 클립. 겨드랑이 높이에 맞추기 |
| 버클 | Buckle | 가랑이 사이 빨간 버튼 결합부 |
| ISOFIX / LATCH | Lower Anchor Connectors | 카시트 뒤의 줄+훅 → 차량 좌석 철봉에 걸어 고정 |
| 수평 화살표 | Level arrow | 카시트 옆면 화살표. 뒤보기 시 바닥과 수평이어야 정상 |

## 1단계: 테슬라 모델3 → 인천공항

**뒤보기 + ISOFIX** (매뉴얼 p.18~22)

1. **발판을 앞쪽으로 접기** — 카시트를 들어서 밑의 발판을 앞으로 접는다  
   *(Flip recline stand toward front to recline position — p.18)*

2. **어깨끈을 아래쪽 구멍에 끼우기** — 4줄 중 아래 3개에서 어깨 높이에 가장 가까운 것 선택  
   *(Harness straps in one of 3 lower slots, at or below shoulders — p.20)*

3. **앞면 천 벗기고, 줄+훅을 앞면 벨트 구멍으로 통과시키기** — 카시트 앞면의 천(시트패드)을 벗기면 양쪽에 벨트 통로가 보임. 카시트 뒤에 달린 줄+훅을 이 통로로 통과  
   *(Route LATCH strap through rear-facing belt path — p.21)*

4. **차량 좌석 철봉에 훅 걸기** — 뒷좌석 쿠션 사이에 숨어있는 철봉(ISOFIX bar)을 찾아 좌우 훅을 각각 걸고 세게 당겨보기  
   *(Attach hooks to vehicle lower anchor bars, pull hard to confirm — p.21)*

5. **몸으로 눌러서 줄 조이기** — 카시트 위에 체중을 실어 누르면서 동시에 LATCH 줄을 당겨 조임. 카시트를 잡고 흔들어볼 때 좌우/전후로 **2.5cm 이상 움직이면 안 됨**  
   *(Push down with full weight, pull strap tight. Must not move >25mm — p.22)*

6. **카시트 옆면 화살표 확인** — 화살표가 바닥과 수평이어야 정상. 기울어져 있으면 **카시트 앞쪽 아래에 말아둔 수건을 넣어** 각도 조정  
   *(Level arrow must be parallel to ground. Use rolled towel if needed — p.18)*

> ⚠️ **절대 위쪽 슬롯(맨 위 구멍)은 뒤보기에 쓰지 마세요** *(Do NOT use upper harness slots rear-facing)*  
> ⚠️ **ISOFIX 줄+훅과 차량 시트벨트를 동시에 쓰지 마세요** *(Never use LATCH and seat belt together)*

### 수건 받치기 상세 설명
카시트를 뒤보기로 설치하면 옆면에 수평 화살표(Level Arrow)가 있어요. 이 화살표가 바닥과 수평이어야 아기 머리가 자연스럽게 뒤로 기대지는 각도가 돼요.

차량 좌석마다 기울기가 달라서, 발판을 접어도 화살표가 수평이 안 되는 경우가 있어요. **카시트가 너무 세워져 있으면 아기 머리가 앞으로 꺾여서 기도가 좁아질 수 있어요.**

해결법: **카시트 앞쪽(발 쪽) 아래에 단단히 말은 수건이나 담요를 받쳐서** 앞부분을 살짝 들어올림 → 뒤쪽이 더 눕혀져서 화살표가 수평에 가까워짐.

- 화살표가 수평이면 → 그대로 OK
- 화살표가 앞으로 기울어져 있으면 → 수건 끼우기
- 테슬라와 푸조 좌석 각도가 다르니 **둘 다 설치 후 확인**

## 2단계: SAS 비행기 (ICN→CPH→GVA / 귀국편도 동일)

**앞보기 + 항공기 벨트** (매뉴얼 p.49~50)

1. **발판을 뒤쪽으로 접기** — 카시트 밑 발판을 뒤로 접어서 카시트가 꼿꼿하게 서도록  
   *(Flip recline stand toward rear to upright position — p.30)*

2. **어깨끈을 위쪽 구멍으로 옮기기** — 4줄 중 위 3개에서 어깨 높이에 가장 가까운 것 선택  
   *(Harness straps in one of 3 upper slots, at or above shoulders — p.31, 35)*

3. **항공기 좌석을 완전히 세우고, 카시트를 앞보기로 올리기**  
   *(Aircraft seat fully upright, place car seat forward-facing — p.49)*

4. **벨트를 뒷면 구멍으로 통과** — 카시트 뒷면 시트패드를 들추면 벨트 통로가 보임. 항공기 벨트를 한쪽으로 넣어서 반대쪽으로 빼냄  
   *(Thread aircraft belt through forward-facing belt path — p.49)*

5. **버클 채우고 벨트 최대한 조이기**  
   *(Fasten belt buckle and pull tight)*

6. **여분 벨트를 다시 구멍으로 통과시켜 추가로 조임** — 카시트 위에 몸을 실어 누르면서 남은 벨트를 당겨 조임. 흔들어볼 때 2.5cm 이상 움직이면 다시 조임  
   *(Route excess belt back through belt path, push down and pull to tighten — p.50)*

> ⚠️ **항공기에서는 ISOFIX/LATCH 절대 사용 금지** *(Never use LATCH on aircraft)*  
> ⚠️ **아래쪽 슬롯(맨 아래 구멍)은 앞보기에 쓰지 마세요** *(Do NOT use lowest harness slot forward-facing)*

### 참고
- 버클이 아이 등 뒤에 오면 작은 베개/담요로 쿠션 대주기 (p.50)
- **CPH 환승 시 카시트 분리 → 들고 이동 → 다음 비행기에서 동일하게 재설치** (환승 1시간 50분)
- 승무원에게 매뉴얼 보여주면 설치 확인 도움받을 수 있음. *Show this manual to cabin crew for installation check.*

## 3단계: 푸조 5008 (프랑스 현지, 13일간)

**뒤보기 + ISOFIX** — **1단계와 완전히 동일한 방법**

### 푸조 5008 참고사항
- ISOFIX 철봉은 **2열 양쪽 외측 좌석에만** 있음 (가운데 없음)
- 푸조의 ISOFIX는 유럽식이지만 Evenflo Tribute의 LATCH 훅과 호환됨 (같은 철봉 방식)
- 수건 받치기 필요 여부: 설치 후 수평 화살표 확인 (테슬라와 각도 다를 수 있음)

### 13일간 생활 팁
- 카시트는 뒤보기로 고정해두고, 아이만 태우고 내리는 방식으로
- 매회 어깨끈 조임 확인: 어깨 위에서 엄지손가락으로 끈을 잡아봐서 잡히면 안 됨
- 가슴 클립은 겨드랑이 높이
- **여름 주차 시** 버클/금속부분 화상 주의 → 밝은색 담요로 덮어두기

## 4단계: 귀국 (6/11)

| 순서 | 할 일 |
|------|------|
| 1 | 숙소 체크아웃 → 푸조에서 카시트 분리 |
| 2 | GVA 공항에서 가족 하차 → 혁인 혼자 반납 → 셔틀 복귀 |
| 3 | 카시트 들고 탑승 (기내 반입) |
| 4 | 기내: **2단계와 동일** (앞보기 + 항공벨트) |
| 5 | CPH 환승: 카시트 분리 → 들고 이동 → 재설치 (1시간 50분) |
| 6 | ICN 도착 → 테슬라에 **1단계와 동일** (뒤보기 + ISOFIX) |

> ⚠️ **CPH 환승이 가장 바쁜 구간** — WCHR + 아기 + 카시트 + 유모차 동시 이동

## 매회 아이 태울 때 체크리스트

1. ✅ 등+엉덩이가 카시트 바닥/등받이에 평평하게 밀착  
   *(Back and bottom flat against seat surfaces — p.41)*

2. ✅ 어깨끈을 어깨 위로 당김 → 버클 딩 소리 날 때까지 끼우기  
   *(Slide harness over shoulders, insert buckle tongues until click — p.41)*

3. ✅ 어깨끈 조이기 — 카시트 앞면 아래쪽 줄을 당겨서 조임. 어깨 위에서 엄지손가락으로 끈을 잡아봐서 **잡히면 안 됨**  
   *(Pull harness adjuster strap at front to tighten. No slack at shoulders — p.43)*

4. ✅ 가슴 클립을 겨드랑이 높이에 맞추고 딸칵 결합  
   *(Chest clip at armpit level, snap together — p.42)*

5. ✅ 가랑이 벨트가 몸에 밀착되는지 확인 — 엉덩이 아래로 가면 절대 안 됨  
   *(Crotch strap snug, never under bottom — p.42)*
