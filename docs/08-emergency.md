## 프랑스 운전 참고

### 고속도로 톨비
- 프랑스 A40 (White Motorway): 편도 약 €6~8
- 카드/현금 결제 가능
- 톨부스에서 티켓 뽑고 출구에서 결제

### 주유
- 무연 휘발유: **Sans Plomb / SP95 또는 SP98**
- 셀프 주유 기본, 카드 결제
- 디젤(Gazole) 넣지 않도록 주의 — 우리 차는 가솔린 하이브리드

### 속도 제한
| 도로 | 제한속도 |
|------|---------|
| 고속도로 (Autoroute) | 130 km/h (비 올 때 110) |
| 일반도로 | 80 km/h |
| 시내 | 50 km/h |
| 주거지역 | 30 km/h |

### 몽블랑 터널 (쿠르마요르 방면)
- 편도 통행료: ~€50 (2026년 요금 현지 확인)
- Les Houches에서 35분
- 일요일은 대기 가능성 있음 — 여유 시간 확보

### 주차
- Les Houches 숙소: 지하주차장 #11번 (무료)
- 샤모니: 유료 주차 (Grépon 주차장 등)
- Lac de Passy: 주차 €3

## 날씨 참고 (5월 말~6월 초)

| 항목 | 예상 |
|------|------|
| 낮 기온 | 15~22°C |
| 밤 기온 | 5~10°C |
| 고산(에귀뒤미디 3,842m) | -5~5°C |
| 비 | 간헐적 소나기 가능 |
| 일몰 | ~21:00 (해가 길다) |

### 복장 제안
- 레이어링 필수 (아침 시원, 낮 따뜻, 저녁 선선)
- 에귀 뒤 미디: 패딩/플리스 + 선글라스 + 선크림
- 호수/마을: 가벼운 긴팔 + 바람막이
- 아기: 겹겹이 입히기, 모자, 자외선 차단
