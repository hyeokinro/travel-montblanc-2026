## ⚠️ 스위스 입국 시 식품 검역 규정

우리 루트: **한국(비EU) → CPH(EU/쉥겐 입국) → GVA(스위스)**

GVA에서 프랑스 섹터로 나가므로 스위스 세관을 정식으로 통과하지는 않지만, 경유지가 스위스이고 쉥겐 내부에서도 세관 검사가 있을 수 있으므로 규정을 알아두는 게 안전해요.

### 반입 금지 / 주의 품목

| 구분 | 규정 |
|------|------|
| **육류/육가공품** | 비EU국가산 → **원칙적 반입 금지**. 한국산 육포, 소시지, 햄 등 불가 |
| **유제품 (버터/크림)** | 1kg까지만 면세. 초과 시 관세 |
| **과일/채소** (생것) | 비EU국가산 → **반입 금지** (파인애플, 바나나, 코코넛, 대추, 두리안만 예외) |
| **달걀** | 비EU국가산 → 반입 금지 |
| **꿀** | 비EU국가산 → 반입 금지 |

### 반입 가능 (문제 없는 것)

| 품목 | 비고 |
|------|------|
| 고추장/된장/참기름 | 발효식품+식물성 → OK |
| 김치 | 식물성이면 OK. **고기/젓갈 들어간 김치는 리스크** |
| 라면 | OK (건조식품) |
| 김 | OK |
| 쌀 | OK |
| 쌀과자/떡뻥 | OK |
| 멸치/다시팩 | 수산물 → OK (어류는 허용) |

### 핵심 요약

> 🔴 **고기류(육포, 햄, 소시지)와 생과일/생채소는 가져가지 마세요**
>
> 🟢 **양념류, 건조식품, 라면, 김, 쌀, 수산물은 OK**
>
> 🟡 **김치는 식물성(배추+고춧가루)이면 OK, 젓갈 비중이 크면 리스크**

> 참고: 실제로는 CPH에서 쉥겐 입국 후 GVA까지 쉥겐 내부 이동이라 세관 검사를 받을 가능성은 낮지만, 만약 검사 시 금지 품목이 발견되면 압수+벌금 가능.
