## ⚠️ 스위스 입국 시 식품 검역 규정

우리 루트: **한국(비EU) → CPH(쉥겐 입국) → GVA(스위스)**

GVA에서 프랑스 섹터로 나가므로 스위스 세관을 정식 통과하지는 않지만, 경유지가 스위스이고 세관 검사가 있을 수 있으므로 규정을 알아두는 게 안전해요.

### 🔴 반입 금지

| 품목 | 비고 |
|------|------|
| 육류 / 육가공품 | 비EU국가산 → 금지. 육포, 소시지, 햄 등 불가 |
| 생과일 / 생채소 | 비EU국가산 → 금지 (파인애플, 바나나, 코코넛, 대추, 두리안만 예외) |
| 달걀 | 비EU국가산 → 금지 |
| 꿀 | 비EU국가산 → 금지 |
| 유제품 (버터/크림) | 1kg 초과 시 관세 |

### 🟢 반입 가능

| 품목 | 비고 |
|------|------|
| 고추장 / 된장 / 참기름 | 발효+식물성 → OK |
| 라면 | 건조식품 → OK |
| 김 / 쌀 / 쌀과자 | OK |
| 멸치 / 다시팩 | 수산물 → OK |

### 🟡 주의

| 품목 | 비고 |
|------|------|
| 김치 | 식물성(배추+고춧가루)이면 OK. **젓갈 비중 크면 리스크** |

> 실제로는 CPH→GVA가 쉥겐 내부 이동이라 세관 검사 확률은 낮지만, 검사 시 금지 품목 발견되면 압수+벌금 가능.
